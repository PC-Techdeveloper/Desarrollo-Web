Programación Web

La programación Web es una disciplina que se ha popularizado mucho en los últimos año. El crecimiento del internet ha permitido la creación de millones de páginas web nuevas, que ofrecen funcionalidades y servicios a una gran cantidad de usuarios.

Se utilizan 3 principales lenguajes para ella, los cuales son HTML, CSS y JAVASCRIPT. Cada uno cumole con una función específica.

HTML

Este lenguaje le da la estructura a las páginas.

CSS

Este lenguaje es para decorar las páginas.

JAVASCRIPT

Este lenguaje es para manipular elementos de las páginas y darle funcionalidades interactivas.
